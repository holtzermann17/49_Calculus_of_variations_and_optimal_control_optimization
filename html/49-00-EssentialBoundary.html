essential boundary


Let E⊂𝐑n be a measurable set. We define the essential boundary of E as

∂*⁡E:={x∈𝐑n:0<|E∩Bρ⁢(x)|<|Bρ⁢(x)|,∀ρ>0}

where |⋅| is the Lebesgue measure.

Compare the definition of ∂*⁡E with the definition of the topological boundary ∂⁡E which can be written as

∂⁡E={x∈𝐑n:∅⊊E∩Bρ⁢(x)⊊Bρ⁢(x),∀ρ>0}.

Hence one clearly has ∂*⁡E⊂∂⁡E.

Notice that the essential boundary does not depend on the Lebesgue representative of the set E, in the sense that if |E⁢△⁢F|=0 then ∂*⁡E=∂*⁡F. For example if E=𝐐n⊂𝐑n is the set of points with rational coordinates, one has ∂*⁡E=∅ while ∂⁡E=𝐑n.

Nevertheless one can easily prove that ∂*⁡E is always a closed set (in the usual sense).
